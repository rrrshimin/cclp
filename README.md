# CarCare Diary Landing Page

Dark-mode landing page for CarCare Diary, a car maintenance tracking app.

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS v4
- Vite
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

## Notes

- Replace placeholder logo in `Logo.tsx` with your actual logo image
- Replace placeholder app screenshot in `Hero.tsx` with your actual screenshot
- Replace feature images in `Features.tsx` with your actual app screenshots
- Update App Store / Google Play links in `StoreButtons.tsx`
